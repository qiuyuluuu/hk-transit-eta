# 数据源与接口约定

本文记录当前项目使用或计划使用的公开交通接口，以及实现时需要注意的字段语义。

## 九巴 KMB

基础地址：

```text
https://data.etabus.gov.hk/v1/transport/kmb
```

常用接口：

```text
路线列表：
GET /route/

路线详情：
GET /route/{route}/{bound}/{service_type}

路线站点：
GET /route-stop/{route}/{bound}/{service_type}

站点详情：
GET /stop/{stop_id}

到站时间：
GET /eta/{stop_id}/{route}/{service_type}
```

完整 ETA URL 模板：

```text
https://data.etabus.gov.hk/v1/transport/kmb/eta/{stop_id}/{route}/{service_type}
```

### 关键字段

| 字段 | 含义 |
| --- | --- |
| `eta` | 预计到站时间，ISO 8601 字符串 |
| `data_timestamp` | 该 ETA 数据更新时间 |
| `eta_seq` | 同一路线同站点返回的第几班 |
| `service_type` | 九巴服务类型，同一路线可能有多种走线/时段 |
| `seq` | 该站在当前 service type 下的站序 |
| `dest_sc` / `dest_tc` / `dest_en` | 官方目的地显示 |
| `rmk_sc` / `rmk_tc` / `rmk_en` | 备注，例如“原定班次” |

### 实时与原定班次

九巴 ETA 可能同时返回实时预计和预定班次。当前页面规则：

- `rmk_sc` 或 `rmk_tc` 为空：显示为“实时预计”
- `rmk_sc` 或 `rmk_tc` 非空：显示原始备注，例如“原定班次”

这不是项目自行计算的巴士位置，而是九巴官方 ETA 接口提供的到站信息。

## 绿色专线小巴 GMB

基础地址：

```text
https://data.etagmb.gov.hk
```

常用接口：

```text
路线/班次：
GET /route/NT/{route_code}

路线站点：
GET /route-stop/{route_id}/{route_seq}

到站时间：
GET /eta/route-stop/{route_id}/{route_seq}/{stop_seq}
```

小巴实现注意：

- 同一路线代码可能对应多个 `route_id`
- 必须明确 `route_id`、`route_seq`、`stop_seq`
- ETA 可能返回 `enabled=true` 但 `eta=[]`，表示端点有效但当前没有 ETA
- 小巴官方站名与九巴站名不完全一致；页面显示使用九巴站名，内部配置保留小巴官方站点信息
- 小巴 API 当前未返回 `Access-Control-Allow-Origin`，浏览器不能直接跨域请求；页面通过 Cloudflare Pages Function 的同源 `/api/gmb/...` 代理访问

小巴 ETA 字段：

| 字段 | 含义 |
| --- | --- |

| `remarks_sc` / `remarks_tc` / `remarks_en` | 备注，例如“未开出” |

页面显示规则与九巴一致：备注为空时显示“实时预计”，备注存在时显示官方备注。

## 上班出门建议

出门建议复用上班组现有 ETA 结果，不额外请求新接口。当前候选范围：

- 逸珑湾 `PA150`：`272A`、`27A`、`27B`
- 云汇 `PA206`：`272A`、`27A`、`27B`

规则：

1. 预留 5 分钟步行/候车时间。
2. 只考虑 `ETA >= 当前时间 + 5 分钟` 的班次。
3. 按 ETA 时间从早到晚排序。
4. 显示前两项，分别作为“最快方案”和“第二快方案”。
5. 建议出门时间为 `ETA - 5 分钟`。
6. 27A 保留 `快车` / `慢车` 标签；27B 保留 `经大学站` 标签。

## 港铁 MTR

基础地址：

```text
https://rt.data.gov.hk/v1/transport/mtr
```

东铁线实时到站：

```text
GET /getSchedule.php?line=EAL&sta={station_code}
```

当前换乘推荐使用：

| 站点 | station_code | 使用方向 | 默认换乘时间 |
| --- | --- | --- | --- |
| 沙田站 | `SHT` | `UP`，往罗湖/落马洲方向 | 4 分钟 |
| 大学站 | `UNI` | `UP`，往罗湖/落马洲方向 | 3 分钟 |

港铁接口返回 `Access-Control-Allow-Origin: *`，可由浏览器直接请求。

换乘推荐规则：

1. 根据用户手动选择的东铁线位置，取可考虑的换乘站：到大围站时比较沙田站与大学站；到沙田站时比较立即在沙田换乘与继续到大学换乘。
2. 取该站下一班东铁北行列车到站时间；如果用户选择“已到沙田站”，沙田站到站时间视为当前时间。
3. 对每个换乘站，只考虑 `巴士/小巴 ETA >= 东铁到站时间 + 该站默认换乘时间` 的班次。
4. 推荐换乘等待时间最短的方案；等待时间按向上取整显示。

为了避免把用户赶不上的列车当作候选，页面会做下限校正：大围到沙田至少按约 3 分钟，沙田到大学至少按约 7 分钟。
